## ADDED Requirements

### Requirement: 색칠하기 모드 진입 버튼
모드 선택 화면에 색칠하기 모드 진입 버튼이 추가되어야 한다. 기존 모드 버튼들과 동일한 스타일을 따른다.

#### Scenario: 색칠하기 버튼 표시
- **WHEN** 사용자가 모드 선택 화면을 열면
- **THEN** 기존 모드 목록에 색칠하기 모드 버튼이 표시된다

#### Scenario: 색칠하기 모드 진입
- **WHEN** 사용자가 색칠하기 모드 버튼을 탭하면
- **THEN** 색칠하기 화면(`ColoringScreen`)으로 이동한다
