## ADDED Requirements

### Requirement: 포인트별 색상 저장
무지개 붓으로 그린 획은 각 포인트에 개별 색상을 저장해야 한다. 색상은 포인트 추가 시점의 경과 시간을 기반으로 HSL 색상환을 계산하여 결정된다. 한 획 내에서도 포인트마다 색상이 달라질 수 있다.

#### Scenario: 천천히 그릴 때 색상 그라데이션
- **WHEN** 사용자가 무지개 붓으로 10초에 걸쳐 긴 선을 그리면
- **THEN** 획의 시작은 빨강, 끝은 보라 계열로 전체 색상환을 한 바퀴 순환한 그라데이션으로 렌더링된다

#### Scenario: 빠르게 그릴 때 비슷한 색상
- **WHEN** 사용자가 무지개 붓으로 1초 미만에 짧은 선을 그리면
- **THEN** 획이 단일 색조에 가까운 색으로 렌더링된다

#### Scenario: 저장 후 불러오기 색상 일치
- **WHEN** 무지개 붓으로 그린 그림을 저장하고 다시 불러오면
- **THEN** 각 포인트의 색상이 저장 시점과 동일하게 복원된다

### Requirement: 인접 포인트 간 색상 보간 렌더링
시스템은 인접 포인트 사이를 시작 색상에서 끝 색상으로 선형 보간하여 부드러운 그라데이션 세그먼트로 렌더링해야 한다. 색상 경계가 뚜렷하게 보여서는 안 된다.

#### Scenario: 세그먼트 경계 부드러움
- **WHEN** 무지개 붓 획을 렌더링하면
- **THEN** 인접한 두 포인트 사이 선분이 두 포인트의 색상 사이를 부드럽게 이어주는 그라데이션으로 그려진다

### Requirement: 소프트 브러시 외곽 흐림
시스템은 무지개 붓 획을 렌더링할 때 `MaskFilter.blur`를 적용하여 획의 외곽선이 부드럽게 흐려지도록 해야 한다. 흐림 강도(`blurSigma`)는 획 굵기에 비례한 기본값을 사용하며 나중에 조정 가능하도록 저장된다.

#### Scenario: 외곽 흐림 확인
- **WHEN** 무지개 붓으로 선을 그리면
- **THEN** 획의 중심은 진하고 외곽으로 갈수록 투명하게 흐려지는 소프트 브러시 효과가 나타난다

#### Scenario: blurSigma 저장 및 복원
- **WHEN** 무지개 붓 획이 저장되고 불러와질 때
- **THEN** `blurSigma` 값이 보존되어 동일한 흐림 강도로 렌더링된다

### Requirement: 10초 주기 색상 사이클
시스템은 획 시작 시점을 기준으로 경과 시간 10초마다 색상환(hue 0°→360°)을 한 바퀴 순환해야 한다. 사이클은 무한 반복된다.

#### Scenario: 10초 후 색상 반복
- **WHEN** 사용자가 무지개 붓으로 10초 이상 연속으로 그리면
- **THEN** 10초 시점의 색상이 시작 색상(빨강)으로 돌아온다
