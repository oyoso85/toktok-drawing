## ADDED Requirements

### Requirement: 대칭축 표시
대칭 그리기 모드 진입 시 캔버스 중앙에 대칭축(세로선)을 표시해야 한다(SHALL).

#### Scenario: 모드 진입 시 대칭축 표시
- **WHEN** 사용자가 대칭 그리기 모드에 진입한다
- **THEN** 캔버스 세로 중앙에 점선 형태의 대칭축이 표시된다

### Requirement: 실시간 대칭 미러링
사용자가 한쪽에 그리면 대칭축 기준으로 반대쪽에도 실시간으로 동일한 스트로크가 그려져야 한다(SHALL).

#### Scenario: 왼쪽에 그리면 오른쪽에 미러링
- **WHEN** 사용자가 대칭축 왼쪽에 스트로크를 그린다
- **THEN** 대칭축 기준으로 오른쪽에 동일한 스트로크가 실시간으로 미러링되어 그려진다

#### Scenario: 오른쪽에 그리면 왼쪽에 미러링
- **WHEN** 사용자가 대칭축 오른쪽에 스트로크를 그린다
- **THEN** 대칭축 기준으로 왼쪽에 동일한 스트로크가 실시간으로 미러링되어 그려진다

#### Scenario: 대칭축을 걸쳐 그리기
- **WHEN** 사용자가 대칭축을 가로질러 스트로크를 그린다
- **THEN** 전체 스트로크와 그 미러링이 모두 표시된다

### Requirement: 드로잉 도구 사용
대칭 그리기 모드에서도 색상, 브러시 크기 등 드로잉 도구를 사용할 수 있어야 한다(SHALL). 미러링된 스트로크에도 동일한 도구 설정이 적용된다.

#### Scenario: 색상 변경 후 대칭 그리기
- **WHEN** 사용자가 빨간색을 선택하고 스트로크를 그린다
- **THEN** 원본과 미러링된 스트로크 모두 빨간색으로 표시된다

### Requirement: 실행 취소
실행 취소 시 원본 스트로크와 미러링된 스트로크가 함께 제거되어야 한다(SHALL).

#### Scenario: 대칭 스트로크 실행 취소
- **WHEN** 사용자가 실행 취소 버튼을 탭한다
- **THEN** 마지막 원본 스트로크와 그에 대응하는 미러링 스트로크가 함께 제거된다
