## ADDED Requirements

### Requirement: 가이드 선 템플릿 표시
선 따라 그리기 모드 진입 시 내장된 가이드 선 템플릿 목록을 표시하고, 사용자가 선택하면 캔버스에 가이드 선을 렌더링해야 한다(SHALL).

#### Scenario: 템플릿 목록 표시
- **WHEN** 사용자가 선 따라 그리기 모드에 진입한다
- **THEN** 앱에 내장된 가이드 선 템플릿 목록이 썸네일로 표시된다

#### Scenario: 템플릿 선택 후 캔버스 표시
- **WHEN** 사용자가 템플릿을 선택한다
- **THEN** 캔버스에 가이드 선이 연한 색상으로 표시되고 드로잉 도구가 활성화된다

### Requirement: 가이드 선 위에 따라 그리기
사용자는 가이드 선 위에 색상이나 패턴을 선택하여 손가락으로 따라 그을 수 있어야 한다(SHALL). 사용자의 스트로크는 가이드 선 위에 겹쳐서 표시된다.

#### Scenario: 색상 선택 후 따라 그리기
- **WHEN** 사용자가 색상을 선택하고 가이드 선 위를 손가락으로 긋는다
- **THEN** 선택한 색상으로 스트로크가 가이드 선 위에 그려진다

#### Scenario: 패턴 선택 후 따라 그리기
- **WHEN** 사용자가 패턴(점선, 별 등)을 선택하고 가이드 선 위를 긋는다
- **THEN** 선택한 패턴으로 스트로크가 가이드 선 위에 그려진다

### Requirement: 가이드 선 시각적 피드백
가이드 선과 사용자의 스트로크가 시각적으로 구분되어야 한다(SHALL).

#### Scenario: 가이드 선과 사용자 스트로크 구분
- **WHEN** 사용자가 가이드 선 위에 그린다
- **THEN** 가이드 선은 연한 회색/점선으로, 사용자 스트로크는 선택한 색상/패턴으로 표시된다

### Requirement: 실행 취소 및 지우기
사용자는 마지막 스트로크를 취소하거나 전체를 지울 수 있어야 한다(SHALL).

#### Scenario: 실행 취소
- **WHEN** 사용자가 실행 취소 버튼을 탭한다
- **THEN** 가장 최근에 그린 스트로크가 제거된다

#### Scenario: 전체 지우기
- **WHEN** 사용자가 전체 지우기 버튼을 탭한다
- **THEN** 사용자가 그린 모든 스트로크가 제거되고 가이드 선만 남는다
