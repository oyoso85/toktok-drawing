## ADDED Requirements

### Requirement: 색상 팔레트
모든 드로잉 모드에서 사용할 수 있는 색상 팔레트를 제공해야 한다(SHALL). 3~7세 아이가 쉽게 선택할 수 있도록 큰 색상 원형 버튼으로 구성한다.

#### Scenario: 색상 팔레트 표시
- **WHEN** 드로잉 화면이 활성화된다
- **THEN** 하단에 기본 색상 팔레트(빨강, 주황, 노랑, 초록, 파랑, 보라, 분홍, 갈색, 검정, 흰색 등)가 원형 버튼으로 표시된다

#### Scenario: 색상 선택
- **WHEN** 사용자가 색상 버튼을 탭한다
- **THEN** 해당 색상이 활성화되고 선택된 색상 버튼에 시각적 표시(테두리 등)가 나타난다

### Requirement: 브러시 크기 조절
사용자는 스트로크의 굵기를 조절할 수 있어야 한다(SHALL). 3~7세 대상이므로 슬라이더 대신 3~4단계 크기 버튼을 제공한다.

#### Scenario: 브러시 크기 선택
- **WHEN** 사용자가 크기 버튼(작은/중간/큰/매우 큰)을 탭한다
- **THEN** 이후 그리는 스트로크에 선택한 굵기가 적용된다

#### Scenario: 크기 미리보기
- **WHEN** 크기 버튼이 표시된다
- **THEN** 각 버튼에 해당 크기를 시각적으로 보여주는 원형 미리보기가 포함된다

### Requirement: 지우개
사용자는 지우개 도구로 특정 부분을 지울 수 있어야 한다(SHALL).

#### Scenario: 지우개 사용
- **WHEN** 사용자가 지우개 도구를 선택하고 캔버스를 문지른다
- **THEN** 해당 영역의 스트로크가 지워진다

### Requirement: 도구 바 레이아웃
드로잉 도구 바는 화면 하단에 고정 배치되며, 캔버스 영역을 최대한 확보해야 한다(SHALL).

#### Scenario: 도구 바 위치
- **WHEN** 드로잉 화면이 표시된다
- **THEN** 도구 바가 화면 하단에 고정되며 캔버스는 나머지 영역을 차지한다

#### Scenario: 도구 바 구성
- **WHEN** 도구 바가 표시된다
- **THEN** 색상 팔레트, 도구 선택(펜/붓/지우개), 크기 조절, 실행 취소/다시 실행 버튼이 포함된다
