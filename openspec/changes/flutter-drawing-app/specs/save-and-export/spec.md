## ADDED Requirements

### Requirement: 그림 로컬 저장
사용자는 현재 드로잉을 기기 내부에 저장할 수 있어야 한다(SHALL). 저장 시 드로잉 데이터(스트로크 정보)와 메타데이터(모드, 생성일, 썸네일)를 함께 저장한다.

#### Scenario: 그림 저장
- **WHEN** 사용자가 저장 버튼을 탭한다
- **THEN** 현재 드로잉이 기기에 저장되고 저장 완료 피드백이 표시된다

#### Scenario: 자동 저장
- **WHEN** 사용자가 드로잉 화면에서 모드 선택 화면으로 돌아간다
- **THEN** 현재 작업 중인 드로잉이 자동으로 저장된다

### Requirement: 갤러리 (내 그림 목록)
저장된 그림 목록을 갤러리 형태로 조회할 수 있어야 한다(SHALL).

#### Scenario: 갤러리 진입
- **WHEN** 사용자가 모드 선택 화면에서 갤러리 버튼을 탭한다
- **THEN** 저장된 그림들이 썸네일 그리드로 표시된다

#### Scenario: 저장된 그림 열기
- **WHEN** 사용자가 갤러리에서 그림 썸네일을 탭한다
- **THEN** 해당 그림이 원래 모드의 드로잉 화면에서 열린다

#### Scenario: 저장된 그림 삭제
- **WHEN** 사용자가 갤러리에서 그림을 길게 눌러 삭제를 선택한다
- **THEN** 해당 그림이 기기에서 삭제된다

### Requirement: 이미지 내보내기
사용자는 드로잉을 PNG 이미지로 내보낼 수 있어야 한다(SHALL).

#### Scenario: 이미지로 내보내기
- **WHEN** 사용자가 내보내기 버튼을 탭한다
- **THEN** 현재 드로잉이 PNG 이미지로 변환되어 기기 갤러리(사진 앱)에 저장된다

### Requirement: 그림 공유
사용자는 내보낸 이미지를 다른 앱으로 공유할 수 있어야 한다(SHALL).

#### Scenario: 공유 기능
- **WHEN** 사용자가 공유 버튼을 탭한다
- **THEN** OS 공유 시트가 표시되어 메시지, 메일 등 다른 앱으로 이미지를 공유할 수 있다
