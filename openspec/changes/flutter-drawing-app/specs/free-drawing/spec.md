## ADDED Requirements

### Requirement: 빈 캔버스 제공
자유 그리기 모드 진입 시 빈 흰색 캔버스를 제공해야 한다(SHALL).

#### Scenario: 모드 진입 시 빈 캔버스
- **WHEN** 사용자가 자유 그리기 모드에 진입한다
- **THEN** 빈 흰색 캔버스와 드로잉 도구 바가 표시된다

### Requirement: 다양한 드로잉 도구 지원
펜, 색연필, 붓, 패턴 등 다양한 드로잉 도구로 자유롭게 그릴 수 있어야 한다(SHALL).

#### Scenario: 펜 도구로 그리기
- **WHEN** 사용자가 펜 도구를 선택하고 캔버스에 손가락으로 긋는다
- **THEN** 일정한 굵기의 선명한 선이 그려진다

#### Scenario: 붓 도구로 그리기
- **WHEN** 사용자가 붓 도구를 선택하고 캔버스에 손가락으로 긋는다
- **THEN** 필압에 따라 굵기가 변하는 부드러운 스트로크가 그려진다

#### Scenario: 색연필 도구로 그리기
- **WHEN** 사용자가 색연필 도구를 선택하고 캔버스에 손가락으로 긋는다
- **THEN** 연필 질감의 반투명 스트로크가 그려진다

### Requirement: 배경색 변경
사용자는 캔버스 배경색을 변경할 수 있어야 한다(SHALL).

#### Scenario: 배경색 변경
- **WHEN** 사용자가 배경색 변경 옵션에서 색상을 선택한다
- **THEN** 캔버스 배경이 선택한 색상으로 변경되며 기존 스트로크는 유지된다

### Requirement: 실행 취소 및 다시 실행
사용자는 스트로크 단위로 실행 취소와 다시 실행을 할 수 있어야 한다(SHALL).

#### Scenario: 실행 취소
- **WHEN** 사용자가 실행 취소 버튼을 탭한다
- **THEN** 가장 최근 스트로크가 제거된다

#### Scenario: 다시 실행
- **WHEN** 사용자가 실행 취소 후 다시 실행 버튼을 탭한다
- **THEN** 취소된 스트로크가 복원된다

#### Scenario: 전체 지우기
- **WHEN** 사용자가 전체 지우기 버튼을 탭한다
- **THEN** 모든 스트로크가 제거되고 빈 캔버스 상태로 돌아간다
